Our goal

To create a loan pre-approval tool with a high level of accuracy using a machine learning model. The tool must be able to be implemented in real time, accessible via the internet, and use minimal data inputs.


About the data

We selected a dataset from Kaggle that we felt had enough data points to train and test a machine learning model. This dataset came from Credit One Bank and included data from 5,000 loans and included 20 attributes of each loan as well as whether the loan had defaulted. The attributes contained information about the borrower, details of the loan structure, and historical information between the borrower and the bank. The data did not include any classifications of a borrower’s credit profile or FICO scores.


Data preprocessing

While processing the data prior to implementing the machine learning models, we noticed that several the attributes that were unclear or seemed to be categorized without specific data on the classification. We removed these attributes from the dataset. We also categorized the remaining attributes that were in a numerical format including age, loan amount, and loan duration. We also chose to remove other attributes that were related to the relationship between the bank and the borrower. We felt that using attributes that were inclusive of only the borrower and requested loan could help to remove any bias that the model may have regarding current bank customers as the data on them would be more robust.


In the end we reduced the number of attributes used in the model from 20 to 9 including:
- Loan term
- Loan purpose
- Loan amount
- Years at current employer
- Co-borrower or guarantor
- Age of applicant
- Rent or Own home
- Active telephone service
- Foreign worker


Machine Learning models

K-Nearest Neighbors (KNN) - 88.1% Accuracy

K-Nearest Neighbors algorithm is the simplest of all the supervised machine learning algorithms. That classifies new cases based on the similarity measures (with regard to distance functions). It simply calculates the distance of a new data point to all other training data points. The distance can be of any type e.g Euclidean or Manhattan etc. It then selects the K-nearest data points, where K can be any integer. Finally, it assigns the data point to the class to which the majority of the K data points belong.

K-Nearest Neighbors computes the likelihood of each data point defaulting or not defaulting on a loan. For example, customer 1 has the probability of not defaulting of 100%. For customer 2, the likelihood of not defaulting is 97%. For customer 3 it is 73%. For customer 4 the likelihood of defaulting is 53% which means that this customer or people like that are likely to default.



Support Vector Machine (SVM) - 93.5% Accuracy

A Support Vector Machine (SVM) is a supervised machine learning algorithm which can be used for both classification and regression problems.



Deep Neural Network (DNN) - 93.7% Accuracy

A neural network is an unsupervised machine learning model built to simulate the activity of the human brain. Its primary use is for pattern recognition and this is achieved by the passing input through various layers of connections that simulate neural paths of a brain. The simplest neural network will have an input layer, an output layer, and a single hidden layer between. Each layer performs specific types of sorting and ordering in a process that is referred to as feature hierarchy. When training the model, this will add different weights specific to input data and attempt to find patterns in the data that will result in the most accurate prediction of the data output.

The model that we used was considered a “deep learning” neural network, which is classified as a neural network with more than 1 hidden layer. After testing different structures, we chose to use 3 hidden layers in our model with 51 nodes in each layer and set the model to run for 100 epochs.



Feature importance

We used a Lasso model to determine which of the impacts had the highest amount of influence on whether a loan would or would not default. The loan term and loan amount were the most relevant factors in the predicting a default.
