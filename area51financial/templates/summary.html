Project Summary

Our goal was to use a machine learning model to accurately predict if a loan would default given a minimal set of inputs. We used three different machine learning models to see which one could generate the most accurate model. In the end the accuracy of the SVM model and the DNN model were very close at 93.5% accuracy and 93.7% accuracy respectively.

We wanted also to determine which model had the highest accuracy specifically to predicting that a loan would not default. If the model would predict that a loan would default (even if it was incorrect and the loan would not default) this would still be ok for the bank as they would not lose their capital, only the opportunity for profits of a loan that wasnâ€™t extended. This scenario would be described as a false negative. If the model would predict a loan would not default, but in the end it would actually default (false positive), this would be very problematic for the bank as they could lose the capital investment in the loan.

We created confusion matrices for each of the models to determine which model would provide the lowest level of false positives. All three models were close with KNN at 12.4%, SVM at 13.5%, and DNN at 12.0%. In the end we selected the Deep Neural Network to run our application. Although the 12.0% potential failure rate of pre-approved loans is likely to high for a bank to incorporate into their application processes, it could still be used as somewhat of a filter to expedite the consumer lending process. We could also work to fine tune each model in the future and hopefully with more data be able to be more accurate with our predictions.
